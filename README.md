## üéº How It Works - The Complete Pipeline

Grammy Engine orchestrates a sophisticated AI pipeline that transforms your text prompt into a professional track:

### **Stage 1: Prompt Intelligence** (2-3 seconds)
```python
Input: "Chill lo-fi beat for studying, 85 BPM, vinyl crackle"
         ‚Üì
GPT-4 Enhancement:
  - Genre Analysis: Lo-Fi Hip Hop, Chillhop
  - Mood Extraction: Calm, focused, nostalgic
  - Instrumentation: Rhodes piano, jazz guitar, laid-back drums
  - Technical Specs: -18 LUFS, warm analog processing
  - Reference Style: "Nujabes, J Dilla influence"
```

**What Happens:**
- AI analyzes emotional intent and context
- Extracts musical parameters (key, tempo, structure)
- Identifies genre conventions and requirements
- Generates detailed technical specifications
- Creates expanded prompt for composition engine

### **Stage 2: Composition Generation** (30-45 seconds)
```python
MusicGen AI Model:
  - Generates instrumental arrangement
  - Applies genre-specific patterns
  - Creates harmonic progressions
  - Designs melodic themes
  - Structures song sections (intro, verse, chorus, etc.)
```

**Advanced Features:**
- **Multi-track Generation:** Separate instruments for clean mixing
- **Style Transfer:** Apply characteristics of reference tracks
- **Structural Intelligence:** Intro, build-ups, drops, outros
- **Harmonic Analysis:** Key detection and chord progressions
- **Tempo Mapping:** Dynamic BPM changes and time signatures

### **Stage 3: Vocal Processing** (10-15 seconds) *[Optional]*
```python
So-VITS-SVC / DiffSinger:
  - AI vocal generation or voice cloning
  - Natural phrasing and breath control
  - Pitch-perfect tuning with human-like variations
  - Dynamic expression and emotion
  - Multiple vocal layers (lead, harmonies, backgrounds)
```

**Capabilities:**
- Generate original AI voices in any style
- Clone existing voices (with permission)
- Morph between different vocal timbres
- Add realistic vibrato, runs, and ornaments
- Create full choir arrangements

### **Stage 4: Mixing** (5-10 seconds)
```python
Automated Mixing AI:
  - EQ: Frequency balancing across all tracks
  - Compression: Dynamic range control
  - Reverb: Spatial depth and ambience
  - Delay: Timing-based effects
  - Stereo Imaging: Width and positioning
  - Automation: Volume, pan, and effect changes
```

**Professional Techniques Applied:**
- Parallel compression for punch
- Multiband EQ for clarity
- Sidechain compression for groove
- Harmonic enhancement for warmth
- Stereo widening for immersion

### **Stage 5: Mastering** (3-5 seconds)
```python
Matchering AI:
  - Final EQ and tonal balance
  - Limiting and loudness optimization
  - Stereo enhancement
  - Quality control checks
  - Format optimization (streaming, radio, film)
```

**Platform Optimization:**
- **Spotify:** -14 LUFS, true peak -1 dB
- **YouTube:** -13 to -15 LUFS
- **Apple Music:** -16 LUFS (Sound Check)
- **Radio:** -10 LUFS, competitive loudness
- **Film/TV:** -20 to -23 LUFS, wide dynamic range

### **Stage 6: Grammy Meter‚Ñ¢ Analysis** (2-3 seconds)
```python
AI Hit Prediction Model:
  - Analyzes 50+ musical features
  - Compares to 10,000+ hit songs database
  - Predicts commercial success probability
  - Identifies strengths and weaknesses
  - Generates actionable recommendations
```

**Analysis Categories:**
1. **Production Quality** (25%)
   - Mix clarity and balance
   - Frequency spectrum analysis
   - Dynamic range evaluation
   - Technical excellence score

2. **Commercial Appeal** (30%)
   - Hook catchiness
   - Genre-market fit
   - Playlist potential
   - Demographic targeting

3. **Innovation** (15%)
   - Unique elements
   - Creative sound design
   - Genre-bending approach
   - Originality score

4. **Emotional Impact** (20%)
   - Listener engagement
   - Mood consistency
   - Energy progression
   - Emotional resonance

5. **Radio Readiness** (10%)
   - Industry standards compliance
   - Loudness optimization
   - Song structure conventions
   - Format compatibility

## üîç Troubleshooting

### **Common Issues**

#### 1. Generation Takes Too Long (>5 minutes)
**Causes:**
- CPU-only system (no GPU acceleration)
- Insufficient RAM (<16GB)
- Model not cached (first run)

**Solutions:**
```bash
# Check GPU availability
docker exec -it grammy-backend nvidia-smi

# Monitor resource usage
docker stats

# Pre-download models
docker exec -it grammy-backend python -c "from services.musicgen_service import download_models; download_models()"

# Increase worker timeout
export GENERATION_TIMEOUT=600
```

#### 2. Out of Memory Errors
**Solutions:**
```bash
# Reduce batch size
export MUSICGEN_BATCH_SIZE=1

# Use smaller model
export MUSICGEN_MODEL=facebook/musicgen-small

# Increase Docker memory
# In Docker Desktop: Settings > Resources > Memory > 8GB+
```

#### 3. Poor Audio Quality
**Check:**
- Prompt clarity (be specific!)
- Model selection (try larger model)
- Sample rate settings
- Mastering configuration

```bash
# Test with high-quality settings
curl -X POST http://localhost:8000/api/generate \
  -d '{
    "prompt": "Your prompt here",
    "quality": "maximum",
    "sample_rate": 48000,
    "bit_depth": 24
  }'
```

#### 4. Authentication Issues
```bash
# Generate new JWT token
curl -X POST http://localhost:8000/api/auth/login \
  -d '{"email": "your@email.com", "password": "yourpassword"}'

# Test token
curl -H "Authorization: Bearer YOUR_TOKEN" \
  http://localhost:8000/api/user/profile
```

#### 5. Database Connection Errors
```bash
# Check Postgres status
docker-compose ps postgres

# Check database logs
docker-compose logs postgres

# Reset database
docker-compose down -v
docker-compose up -d postgres
docker exec -it grammy-backend alembic upgrade head
```

## üí° Prompt Engineering Guide

Master the art of creating perfect prompts for Grammy-worthy results.

### **The Perfect Prompt Formula**

```
[Genre] + [Tempo] + [Mood/Emotion] + [Key Instruments] + [Use Case] + [Style Reference]
```

### **Excellent Prompt Examples**

#### ‚úÖ Highly Effective Prompts

**Example 1: Commercial Pop**
```
"Upbeat dance-pop track, 128 BPM, euphoric and energetic, 
featuring bright synths, punchy drums, catchy vocal hooks, 
perfect for summer festival anthems, similar to Calvin Harris"
```
**Why it works:** Specific genre, exact BPM, clear mood, named instruments, use case, style reference

**Example 2: Lo-Fi Study Music**
```
"Chill lo-fi hip hop beat, 75-85 BPM, calm and focused, 
Rhodes piano, vinyl crackle, jazz guitar, rain sounds, 
for studying and concentration, inspired by Nujabes"
```
**Why it works:** Genre-specific details, tempo range, mood for use case, texture elements

**Example 3: Cinematic Trailer**
```
"Epic orchestral hybrid trailer music, 140 BPM half-time, 
heroic and inspiring, full orchestra with modern percussion, 
massive drums, soaring strings, powerful brass, choir, 
building to climactic finale, for action movie trailers"
```
**Why it works:** Specific sub-genre, complex tempo description, detailed instrumentation, clear purpose

**Example 4: Electronic Dance**
```
"Progressive house track, 126 BPM, uplifting and melodic, 
warm supersaw leads, deep rolling bass, crisp percussion, 
emotional breakdown with piano, festival-ready drop, 
style of Above & Beyond or Lane 8"
```
**Why it works:** Sub-genre specified, exact BPM, emotional arc described, production details

### **Common Prompt Mistakes**

#### ‚ùå Too Vague
```
"Make me a good song"
"Something catchy"
"Cool beat"
```
**Problem:** No direction for AI to follow. Results will be random.

#### ‚ùå Conflicting Requirements
```
"Relaxing and energetic dance track"
"Sad happy song"
"Slow-paced uptempo beat"
```
**Problem:** Contradictory instructions confuse the AI.

#### ‚ùå Too Complex/Mixed Genres
```
"Country rock jazz hip hop dubstep classical fusion 
with trap drums and orchestra and guitars"
```
**Problem:** Too many genres create muddy, unfocused results.

#### ‚ùå Unrealistic Expectations
```
"Make it sound exactly like Bohemian Rhapsody"
"Copy Drake's entire style perfectly"
```
**Problem:** Legal/ethical issues and unrealistic precision requirements.

### **Advanced Techniques**

#### **1. Temporal Descriptors**
Describe how the track evolves over time:

```
"Start with minimal piano and rain, gradually add warm bass 
at 30 seconds, introduce drums at 1 minute, build to full 
arrangement at 1:30, emotional climax at 2:00, fade out 
with piano at 2:30"
```

#### **2. Technical Specifications**
Include production details for pros:

```
"Deep house track, 122 BPM, sidechain compression on pads, 
filtered disco samples, analog-style warm bass, -14 LUFS 
for streaming, stereo width emphasis on synths"
```

#### **3. Emotional Journey Mapping**
Map emotional progression:

```
"Begin mysterious and tense (intro), transition to hopeful 
(verse), explode into triumphant joy (chorus), reflect 
intimately (bridge), resolve with powerful victory (finale)"
```

#### **4. Reference Mixing**
Combine multiple artist references:

```
"Production style of Flume, vocal approach of Billie Eilish, 
song structure like Disclosure, emotional depth of James Blake"
```

#### **5. Negative Prompting**
Specify what to avoid:

```
"Tropical house track WITHOUT steel drums, NO cheesy 
synths, avoid predictable drops, skip traditional house piano"
```

### **Genre-Specific Prompt Templates**

#### **Pop Track**
```
"[Energy level] pop song, [BPM] BPM, [emotion],
featuring [instruments], [hook description],
perfect for [playlist type], similar to [artist]"
```

#### **Hip Hop Beat**
```
"[Sub-genre] hip hop beat, [BPM] BPM, [vibe],
[drum style], [melodic elements], [bass type],
suitable for [rapper style], inspired by [producer]"
```

#### **Electronic/EDM**
```
"[Sub-genre] track, [BPM] BPM, [energy/mood],
[synthesis type] leads, [bass description],
[build/drop characteristics], for [setting/use]"
```

#### **Cinematic/Trailer**
```
"[Intensity] cinematic track, [tempo] BPM,
[orchestration details], [hybrid elements],
[emotional arc], perfect for [media type]"
```

#### **Ambient/Chill**
```
"[Atmosphere] ambient music, [tempo] BPM,
[texture descriptions], [minimal/maximal],
for [activity/setting], mood of [reference]"
```

### **Iteration Strategy**

Start broad, then refine:

**Generation 1:** Basic prompt
```
"Chill electronic music for focus"
```

**Generation 2:** Add specifics
```
"Downtempo electronic, 90 BPM, focused and calm,
ambient pads, minimal beats, for deep work"
```

**Generation 3:** Fine-tune details
```
"Downtempo electronica, 90 BPM, focused and meditative,
warm analog pads, subtle broken beats, field recordings,
tape saturation, for deep work sessions, like Boards of Canada"
```

### **Pro Tips**

1. **Be Specific with BPM**
   - ‚úÖ "128 BPM" 
   - ‚ùå "Fast"

2. **Name Actual Instruments**
   - ‚úÖ "Rhodes piano, Moog bass, TR-808 drums"
   - ‚ùå "Keyboard sounds"

3. **Describe the Vibe/Use Case**
   - ‚úÖ "Perfect for yoga classes and meditation"
   - ‚ùå "Just relaxing"

4. **Reference Real Artists/Tracks**
   - ‚úÖ "Similar to Bon Iver's '22, A Million'"
   - ‚ùå "Kind of indie-ish"

5. **Think About the Listener**
   - ‚úÖ "Energizes workout, builds motivation"
   - ‚ùå "Gym music"

6. **Use Production Terminology**
   - ‚úÖ "Sidechain compression, analog warmth, stereo width"
   - ‚ùå "Make it sound professional"
